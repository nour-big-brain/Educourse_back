This is the backend of Educourse, an online learning platform, built with Spring Boot and Java. It provides REST APIs for managing courses, user enrollment, and progress tracking.

Key features include:

Course Management – create, update, and delete courses.

Enrollment & Progress Tracking – manage student registrations and course progress.

Secure Access – role-based authentication and authorization.

API-Driven – serves data to the Angular frontend for a seamless user experience.

This repository contains the backend implementation only.
